## How to Use

1. Create directory `./data`
2. Add one or multiple subdirectories containing *only* MOV files.

Ex:
```
data
    deckName_1
        IMG_4631.MOV
        IMG_4632.MOV
        ...
        IMG_4689.MOV
    anotherDeckName
        IMG_6107.MOV
        IMG_6108.MOV
        ...
        IMG_6163.MOV
```

3. Run `python main.py` to start the pipeline.
